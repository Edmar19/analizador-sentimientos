# analizador-sentimientos
Aplicación web para análisis de sentimientos de comentarios usando Python y Flask

## Descripción
Aplicación web que analiza sentimientos en comentarios usando Python y Flask

## Características
- Clasifica comentarios como positivos, negativos o neutrales
- Interfaz web con Flask
- Genera reportes automáticos

## Tecnologías
- Python
- Flask
- TextBlob
- HTML/CSS

## Instalación
git clone https://github.com/Edmar19/analizador-sentimientos
cd analizador-sentimientos
pip install -r requirements.txt
python app.py
